# McBurgs - Create your own burger, or choose one from our delicious menu and get ready to eat!

Users can enter in their own burger and add it to the menu. Users can also 'Eat' a burger.

# Technology Used
1. Javascript
2. jQuery
3. NodeJS
4. Express.js
5. HTML
6. Bootstrap
7. Handlebars
8. MySQL
